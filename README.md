# Networking Utilities

This repository contains practical networking snippets and reference material for switches, routers, VLANs, and basic firewall concepts.

The goal is to document simple, repeatable patterns that can be adapted in the field.

## Contents

- cisco  
  Basic switch configuration examples and VLAN configuration snippets.

- mikrotik  
  Example RouterOS configuration for a simple routed network.

- general  
  Notes on network diagrams and a troubleshooting checklist.

- docs  
  High level best practices for small to medium networks.

These files are intended as starting points and reminders, not as drop-in production configurations.
